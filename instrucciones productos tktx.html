Cómo usar los productos TKTX correctamente

Pasos para aplicar TKTX:

  1. Preparar bien el área: Antes de usar TKTX, limpia y seca la piel.
  2. Aplicar una capa gruesa: Extiende la crema sin frotar.
  3. Envolver con vinipel negro: Usa varias capas para que el calor abra los poros y se absorba mejor.
  4. Dejar actuar por 1 hora: No retires el plástico antes del tiempo recomendado.
  5. Retirar con un paño húmedo: No uses alcohol para limpiar.
  6. Limpiar solo el rastro de la máquina: No limpiar el stencil ni el área circundante.

Recomendaciones importantes:

  • 💡 Descanso y alimentación previos antes del procedimiento.
  • ⚙ Equipos listos antes de aplicar TKTX.
  • ⚠ No usar en heridas abiertas, quemaduras, alergias o problemas dermatológicos.
  • 🩹 En suturas, esperar 2 años de cicatrización antes de usar TKTX.
  • 🔥 Para prolongar el efecto, usa compresas de agua tibia 5-7 minutos antes.
  • ❌ No usar TKTX crema durante el procedimiento (puede causar lesiones).
  • ✅ Para procedimientos, usar TKTX GEL en lugar de crema.
  • ⚡ El efecto anestésico varía, pero ayuda a reducir el dolor.
  • ❗ Si usas removedor de stencil, perderás gran parte del efecto anestésico.
  • 👨‍🎨 Solo para artistas profesionales.
  • 🚫 Uso externo únicamente.
